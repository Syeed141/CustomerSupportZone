What is JSX, and why is it used?

   
JSX is a syntax which is used in React that looks similar to HTML but which is written inside JavaScript. It makes the UI structure easier to read and write, allowing developers to describe how components should appear.

What is the difference between State and Props?

   
Props are the data which passed from a parent component to a child component and cannot be changed by the child. State is data managed inside a component and can change over time, causing the component to update.

What is the useState hook, and how does it work?

   
The useState() is a hook which allows functional components to have state. It provides a state value and a function to update that value. When the state is updated, React re-renders the component.

How can you share state between components in React?

   
State is usually shared by lifting it to a common parent component and passing it down to other components through props. 

How is event handling done in React?

    
Event handling in React is done by attaching event handlers to elements using the camelCase event names and passing the functions that run when the event occurs.
